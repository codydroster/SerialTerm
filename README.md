## UARTGamepad

UARTGamepad takes input from a USB or Bluetooth game controller and converts it to to a configurable UART data stream.
#### Requirements:

```
Python3  
pygame  
pyserial  
pygtk  
```

#### Run

```
$ python3 main.py
```
